# Mend Breakables
This mod adds the option to restore objects that normally
stay broken in Hollow Knight. Each group of objects can
be toggled from the menu:
- Gramaphones on the trams
- The gramaphone in Queen's Gardens
- The glass jars in the Tower of Love

Also, if you visit a repaired object, it is fully reset,
so if you want it to stay broken, you will have to toggle
the menu setting and then manually break it again.
